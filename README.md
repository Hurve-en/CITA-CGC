# CITA-CGC
AI-Powered Coral Reef Health Monitoring System

## The Innovation
CITA-CGC turns reef monitoring into a community-powered, AI-assisted workflow. Instead of waiting for delayed manual inspections, people can upload underwater photos and instantly generate structured reef health reports that are useful for local decision-makers.

It combines:
- AI image-based reef assessment
- real-time report collection
- accessible monitoring for non-technical users
- localized support for Cebu communities

## Why I Made This Project
I built CITA-CGC to solve a real local challenge in Cebu: reefs are getting damaged faster than they are being reported and acted on. Communities often see damage first, but there is no simple, consistent system that converts those observations into usable, timely reef intelligence.

This project is my attempt to bridge that gap by giving everyday users and authorities a practical digital tool for faster reef protection.

## The Problem I Am Trying To Solve
Cebu’s reefs support fisherfolk livelihoods, biodiversity, and tourism, but they are threatened by:
- coral bleaching
- illegal fishing
- anchor damage
- crown-of-thorns outbreaks
- rising ocean temperatures

The main issue is response speed. By the time reef damage is formally reported through traditional channels, intervention is often late.

CITA-CGC addresses this by making reef reporting immediate, structured, and actionable.
